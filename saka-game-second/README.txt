Если оба устройства (твой ноутбук и другое устройство) подключены к одному Wi-Fi, ты можешь открыть игру по локальной сети.

В VS Code нажми Ctrl+C, чтобы остановить сервер.

Запусти его снова с флагом "host":

Bash
npx vite --host
В терминале появится новая строчка: Network: http://192.168.x.x:5173/.

Возьми второе устройство (телефон или планшет), открой браузер и введи этот IP-адрес (например, http://192.168.1.5:5173).

Важно: ноутбук должен быть включен, а Vite — запущен.
ОТКРЫТЬ ВИЖУАЛ СТУДИО КОД
НАЖАТЬ CTRL+Ё
В ОТКРЫВШЕЙСЯ КОНСОЛИ НАПИСАТЬ npx vite --host
ПЕРЕЙТИ ПО ССЫЛКЕ С ПОМОЩЬЮ CTRL+ПКМ ИЛИ СКОПИРОВАТЬ В БРАУЗЕР
ИГРАТЬ
npx vite
